# videoPlayer-WeiXin
建立一个基于微信小程序的视频播放器，可任意观看视频，解除VIP的困扰。视频源：搭建一个服务器用脚本来进行视频资源的爬取。然后根据微信小程序前端的传入参数，后端进行爬取，然后反馈给前端进行展示。
